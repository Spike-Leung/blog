# GTD笔记

# 什么是GTD

GTD(Getting-Things-Done),就是把所有想做的事情罗列下来，分门别类，哪些是吗上可以做的，哪些是不用做的，哪些是下一步做的等等，让脑子不必去记住这些事，留下更多脑细胞去思考别的事情。

# 操作

# 罗列所有要做的事情

把所有想做的，需要做的事情都罗列出来，例如要不要换牙刷，还书，阅读某些文章，大后天去什么地方吃饭等等，事无巨细。
